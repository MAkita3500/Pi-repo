# PieCommerce

Projecto PHP para a criação de uma solução e-commerce para a venda de produtos de limpeza.

O Modelo de dados compreende as seguintes representações:

Utilizador (dados de utilizador como nome de login e senha)
Cliente (dados de cliente como nome, data de nascimento e outros)
Produto (dados do produto como exemplo nome e tipo)
Entrega (dados do pedido de entrega de um produto a um cliente)
Endereco (dados do endereço de um cliente)

Essa solução usa a estrutura MVC e outras boas práticas de programação.
